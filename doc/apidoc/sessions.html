POST /auth/sign_in
Email authentication.

Errors

Code Description
401 Invalid login credentials. Please try again.
401 A confirmation email was sent to your account at {{email}}.You must follow the instructions in the email before your account can be activated
403 Email already in use

Examples

POST /auth/sign_in
{
  "email": "george@rubygarage.org",
  "password": "password"
}
200
{
  "data": {
    "id": 2,
    "provider": "email",
    "uid": "george@rubygarage.org",
    "name": null,
    "nickname": null,
    "image": null,
    "email": "george@rubygarage.org"
  }
}
