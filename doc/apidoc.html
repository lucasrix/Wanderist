Wanderist API documentation

Resources

Passwords

Resource Description
POST /auth/password Use this route to send a password reset confirmation email to users that registered by email.

Registrations
Users manager

Resource Description
POST /auth Email registration. A verification email will be sent to the email address provided.

Sessions

Resource Description
POST /auth/sign_in Email authentication.
DELETE /auth/sign_out Use this route to end the user's current session. This route will invalidate the user's authentication token.

Token validations

Resource Description
GET /auth/validate_token Use this route to validate tokens on return visits to the client.
